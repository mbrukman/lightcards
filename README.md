lightcards
==========

<img align="center" src="http://odsod.github.io/lightcards/preview.png">

`Lightcards` is a minimalistic flash card software for learning Chinese characters. 

Getting started
---------------

    npm install -g lightcards
    lightcards init
    cd lightcards && lightcards

Have fun!
